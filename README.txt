This is the source code for the Blue Valley server to push to the Journey platform. From the Blue Valley website, users will be redirected to this third party website to fill out customer information. This information will then be sent from this server to the Journey database for use on the mobile application. This project is written in HTML, CSS, C# using Visual Studio Express for Web on a Windows 8.1 VM.
